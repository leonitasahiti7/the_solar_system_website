# The Solar System Website

The goal of this project is to introduce the planets, moons, and other objects in our solar system in an interactive and visual way.

---

# Features

- Home page with a space-themed design
- Individual pages for different sections (About, Gallery, Blog, etc.)
- Responsive layout
- Custom CSS styling
- JavaScript for interactivity
- Images related to space and planets
